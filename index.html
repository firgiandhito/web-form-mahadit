Formulir Pendaftaran Ma'had IT An'amta

Petunjuk penggunaan formulir pendaftaran

  1. Isi semua data yang diperlukan
  2. Isi data dengan sejujur-jujurnya
  3. Setelah data terisi lengkap, tekan tombol 'kirim'
  4. Anda akan diarahkan ke dalam aplikasi whatsapp / whatsapp web
  5. Jika anda mengisi formulir menggunakaan PC browser, izinkan laman browser untuk membuka aplikasi whatsapp
  6. Tekan tombol kirim untuk mengirimkan pesan kepada musyrif
