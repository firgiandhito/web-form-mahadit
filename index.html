Formulir Pendaftaran Ma'had IT An'amta

Petunjuk penggunaan formulir pendaftaran

  1. Mulai dengan membaca Bismillah
  2. Isi semua data yang diperlukan
  3. Isi data dengan sejujur-jujurnya
  4. Setelah data terisi lengkap, tekan tombol 'kirim'
  5. Anda akan diarahkan ke dalam aplikasi whatsapp / whatsapp web
  6. Jika anda mengisi formulir menggunakaan PC browser, izinkan laman browser untuk membuka aplikasi whatsapp
  7. Tekan tombol kirim untuk mengirimkan pesan kepada musyrif


Syarat pendaftaran Mahasantri Ma'had IT An'amta

  1. Muslim taat beribadah (Shalat berjama'ah di masjid, rutin tilawah Qur'an, dll)
  2. Lulusan Pesantren/Madrasah Aliyah (Maksimal 2 tahun lalu)
  3. Tidak merokok atau mengkonsumsi alkohol
  4. Siap mengikuti peraturan Ma'had IT An'amta Syameela serta bersedia menerima sanksi jika melanggar
  5. Lulus tes penerimaan & seleksi calon Mahasantri Baru
